📦 Intranet Desktop

Guia de Instalação Simplificado

1

Baixar os Arquivos

Primeiro, você precisa baixar os arquivos necessários

A

📥 Baixe o Intranet Desktop (ZIP)

Clique no botão verde abaixo para baixar o aplicativo (arquivo ZIP portable):

⬇️ Baixar Intranet Desktop (ZIP)
💡
Não consegue baixar pelo botão?

Peça o arquivo para o setor de TI da sua empresa.

B

📥 Baixe o NAPS2 (Necessário para Scanner)

Se você vai usar scanner, precisa instalar também o NAPS2:

⬇️ Baixar NAPS2 (ZIP)

Link alternativo (instalador): www.naps2.com/download

⚠️
Não usa scanner?

Pode pular este download e ir direto para o Passo 2.

C

📂 Onde o arquivo foi salvo?

O arquivo baixado geralmente fica na pasta Downloads. Para encontrar:

Como encontrar o arquivo
  1. Abra o Explorador de Arquivos (pasta amarela na barra de tarefas)
  2. No menu lateral esquerdo, clique em "Downloads"
  3. Procure o arquivo: Intranet-Desktop-1.0.0-Portable.zip
📁

Pasta Downloads do Windows

Aqui estão os arquivos que você baixou
2

Extrair e Usar o Aplicativo

Como descompactar o arquivo ZIP e abrir o programa

A

📂 Encontre o arquivo ZIP baixado

Vá até a pasta Downloads e encontre o arquivo Intranet-Desktop-1.0.0-Portable.zip

📁

Pasta Downloads → Intranet-Desktop-1.0.0-Portable.zip

B

📦 Extraia o arquivo ZIP

Clique com o botão direito no arquivo ZIP e escolha:

Menu do Botão Direito
📂 Abrir
📦 Extrair Tudo...
📋 Copiar
🗑️ Excluir
  1. Clique com o botão direito no arquivo ZIP
  2. Clique em "Extrair Tudo..."
  3. Escolha onde quer salvar (pode deixar o padrão)
  4. Clique em "Extrair"
💡
Dica

Extraia para uma pasta fixa como C:\Programas\Intranet Desktop ou deixe no Downloads mesmo.

C

▶️ Abra o programa

Dentro da pasta extraída, encontre e dê dois cliques no arquivo:

🖥️
Intranet Desktop.exe
Dê dois cliques para abrir
D

🛡️ Se aparecer aviso de segurança

Na primeira vez, o Windows pode mostrar um aviso. É só clicar em:

  1. "Mais informações"
  2. "Executar assim mesmo"
✅
Isso é normal!

O Windows mostra este aviso porque o aplicativo é novo. É seguro continuar.

E

⭐ Criar atalho na Área de Trabalho (opcional)

Para facilitar, você pode criar um atalho:

  1. Clique com o botão direito no arquivo Intranet Desktop.exe
  2. Clique em "Enviar para"
  3. Clique em "Área de Trabalho (criar atalho)"
🎉
Pronto! Aplicativo configurado!
📄
Vai usar Scanner?

Continue para o Passo 3 para configurar o scanner.

🚀
Não vai usar Scanner?

Pronto! Dê dois cliques no ícone para abrir o aplicativo.

3

Configurar o Scanner

Apenas se você for usar scanner (opcional)

A

📥 Baixe e Extraia o NAPS2

Se ainda não baixou, clique no botão abaixo:

⬇️ Baixar NAPS2 (ZIP)

Link alternativo (instalador): www.naps2.com/download

Depois de baixar, extraia o ZIP da mesma forma que o Intranet Desktop: botão direito → "Extrair Tudo..." → "Extrair"

B

🔍 Abra o NAPS2

Dentro da pasta extraída, dê dois cliques no arquivo NAPS2.exe

📄

Pasta do NAPS2 → NAPS2.exe

Dê dois cliques para abrir
C

➕ Crie um novo perfil

Na janela do NAPS2:

  1. Clique no botão "Perfis" no menu superior
  2. Clique em "Novo"
➕

Clique em "Perfis" → Depois em "Novo"

D

⚙️ Configure o perfil

Preencha as configurações exatamente como está na tabela:

Campo O que digitar/selecionar
Nome do perfil BrotherADF
Driver WIA
Dispositivo Selecione seu scanner na lista
Origem do papel Alimentador
Tamanho da folha A4 (210 x 297 mm)
Resolução 200 dpi
Modo de cor Colorido
🚨
MUITO IMPORTANTE!

O nome do perfil deve ser exatamente BrotherADF

Escreva igual: letra maiúscula B, resto minúsculo, sem espaços.

E

💾 Salve e teste

  1. Clique em OK para salvar o perfil
  2. Coloque um papel no scanner
  3. Selecione o perfil BrotherADF
  4. Clique em Digitalizar
🎉
Se funcionou, parabéns!

Seu scanner está configurado. Agora pode usar no Intranet Desktop.

🔧

Está com Problema?

Veja as soluções para os problemas mais comuns

⚠️ O aplicativo não abre
✅ Tente isso:
  1. Reinicie o computador
  2. Clique com o botão direito no ícone → "Executar como administrador"
  3. Verifique se o antivírus não está bloqueando
⚠️ Scanner não aparece / não funciona
✅ Tente isso:
  1. Verifique se o scanner está ligado
  2. Verifique se o cabo está conectado
  3. Baixe o NAPS2 se ainda não baixou: Baixar NAPS2 (ZIP)
  4. Teste primeiro no próprio NAPS2 antes de usar no sistema
⚠️ Erro: "Perfil BrotherADF não encontrado"
✅ Solução:
  1. Abra o NAPS2
  2. Crie o perfil seguindo o Passo 3 deste manual
  3. Certifique-se que o nome é exatamente BrotherADF
⚠️ Tela branca / Sistema não carrega
✅ Tente isso:
  1. Verifique sua conexão com a internet
  2. Pressione Ctrl + R para recarregar
  3. Feche e abra o aplicativo novamente
⚠️ Impressora não funciona
✅ Tente isso:
  1. Verifique se a impressora está ligada
  2. Teste imprimir uma página pelo Word ou Bloco de Notas
  3. Se não funcionar nem fora do aplicativo, o problema é na impressora
